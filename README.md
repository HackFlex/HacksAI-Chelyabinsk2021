## Установка

Создадим новое окружение и установим все необходимые пакеты:

```console
foo@bar$ python3 -m venv flex
foo@bar$ source flex/bin/activate
(flex) foo@bar$ pip install -r requirements.txt
```

## Запуск

Запуск
```console
(flex) foo@bar$ python main.py
```
